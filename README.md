# Magyarorsz-g-felsz-njellemz-inek-id-soros-t-rk-pi-adatb-zisa
Magyarország felszínjellemzőinek idősoros térképi adatbázisa, amely a tájhasználati és felszínérdességi viszonyok térbeli és időbeli változásait dolgozza fel egységes térinformatikai keretben. A repozitórium kutatási és oktatási célú térképi állományokat és dokumentált feldolgozási lépéseket tartalmaz.
